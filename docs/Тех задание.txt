Задачи
2.2.1. Систематический обзор литературы по сервисам предиктивной аналитики в образовании с формулированием критериев отбора источников, синтезом результатов, 
выявлением ограничений существующих подходов и систем управления обучением и обоснованием новизны НИР.  
2.2.2. Сбор данных системы управления обучением campus.fa.ru, содержащей баллы студентов разных специальностей по множеству аудиторных, самостоятельных, 
контрольных и курсовых работ по англо- и русскоязычным математическим дисциплинам “Анализ данных”, “Математика”, “Цифровая математика”, 
“Цифровая математика на языке R и Excel”, “Компьютерный практикум”. Подготовка данных к статистическому анализу (анализ структуры курсов, обработка пропусков и ошибок, формирование многомерных временных рядов). 
2.2.3. Статистический анализ баллов студентов по данным системы управления обучением: выявление невыполненных работ и причин невыполнения по количеству попыток 
  и затраченному времени, построение плотностей распределения баллов по группе, по потоку, по дисциплине со сравнением основных статистических характеристик, 
изучение влияния выполняемых студентами работ на итоговые баллы и междисциплинарных  связей.
2.2.4. Математическая постановка задач выделения студентов под академическим риском,  прогнозирования баллов студентов по изучаемой дисциплине и пулу 
  математических дисциплин на последующих курсах, формирования подгрупп студентов по схожести траекторий обучения для персонализации консультаций. 
2.2.5. Постановка статистических гипотез о влиянии количества пропусков, выполнения домашних заданий, на результат контрольных и экзаменационных работ, 
существовании значимых различий в распределениях итоговых баллов между русскоязычными и англоязычными группами, влиянии среднего балла группы на 
  внутригрупповую дисперсию баллов, влиянии времени выполнения заданий и количества попыток на выполнение домашних заданий на итоговый балл, 
выявление нарушений академических норм.
2.2.6 Верификация статистических гипотез на подготовленных данных. Построение доверительных интервалов.
2.2.7 Прогноз итоговых баллов по многомерным временным рядам. Выбор метрик и оценка качества прогноза. 
2.2.8 Выявление подгрупп студентов со схожими траекториями обучения для персонализации консультаций. Разработка метрик схожести траекторий. 
  Использование алгоритмов снижения размерности для визуализации траекторий.
2.2.9 Разработка прототипа сервиса управления обучением, визуализирующего аналитику и предоставляющего рекомендации (персонализированные материалы, ранние предупреждения об академическом риске). Тестирование рекомендаций сервиса на подгруппах студентов. 

Результаты
3.1.1. Методология повышения эффективности образовательного процесса за счет сервиса управления обучением. Формализованная математическая модель прогнозирования 
  академического риска на основе многомерных временных рядов. Статистически проверенные гипотезы о влиянии поведенческих факторов студентов на успеваемость. 
  Кластерная аналитика траекторий обучения с интерпретацией причин успешности/отставания.
3.1.2. Снижение количества студентов под академическим риском, объединение подгрупп студентов для персонализации обучения (консультаций, организации дополнительных курсов, мастер-классов и семинаров, ориентированных на выявленный набор проблемных тем). 
  Рекомендации по адаптации сервиса для нематематических дисциплин. Рекомендации по переходу с существующей системы управления обучением на альтернативные цифровые платформы.
3.1.3. Прототип сервиса управления обучением, реализующий предиктивную аналитику п. 3.1.2 и рекомендации.
